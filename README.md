# mcc-mnc-table

Mobile Country Codes (MCC) and  Mobile Network Codes (MNC),  Network Operator ,  Iso , Country code , MCC and MNC combined table in JSON, CSV, XML 


Object Format


`````````
   {
     "network": "A-Mobile",
     "country": "Abkhazia",
     "mcc": "289",
     "iso": "ge",
     "country_code": 7,
     "mnc": "68",
     "mcc_mnc": 28968
   },
`````````
